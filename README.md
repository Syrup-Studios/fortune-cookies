Datapack path

`(Datapackname)/data/(namespace)/fortunes/(fortunename).json`

Example json
```json
{
  "fortune": "Example Bad Fortune",
  "aura": "NEGATIVE",
  "weight": 5,
  "effects": [
    {
      "effect": "minecraft:weakness",
      "duration": 40,
      "amplifier": 0
    },
    {
      "effect": "minecraft:mining_fatigue",
      "duration": 30,
      "amplifier": 0
    }
  ]
}

```
